## 5. LangGraph 특화 부업 아이디어 (15개)

### LangGraph의 경쟁 우위
1. **상태 관리 우수성:** 감사 추적, 컴플라이언스 필요한 비즈니스
2. **멀티 에이전트 조율:** 전문 에이전트 협업 (리서치, 분석, 검증)
3. **Human-in-the-Loop:** AI가 지원하지만 사람이 승인 (법률, 의료, 금융)
4. **장기 실행 워크플로우:** 체크포인팅으로 시간/일 단위 프로세스
5. **조건부 복잡성:** 신뢰도/규칙 기반 분기 로직
6. **순환 개선:** 반복 개선 워크플로우 (콘텐츠 편집, 리서치 검증)

**LangGraph가 이상적인 이유:**
- T2/T3 지원 티켓이 여러 시스템 통합 필요
- 상태 관리로 플랫폼 간 고객 컨텍스트 유지 (Zendesk, Gorgias, Front)
- Human-in-the-loop으로 행동 실행 전 승인 (환불, 주문 취소)
- Supervisor 패턴으로 전문 에이전트 조율 (주문 추적, 재고, 반품)

**기술 구현:**
- Supervisor 에이전트가 티켓을 전문 하위 에이전트로 라우팅
- Shopify, WooCommerce, 배송 API 통합
- 상태 그래프로 티켓 이력 및 고객 데이터 유지
- 신뢰도 < 0.7 시 사람 에이전트로 에스컬레이션

**타겟 고객:**
- 중형 이커머스 비즈니스 (500-5000 주문/일)

- SaaS: 티켓 볼륨 기반 $500-2000/월
- 비용 절감: T2 티켓 해결 시간 70% 단축
